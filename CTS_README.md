# Chase The Split 

Chase The Split is a web app that turns rowing workout data into usable analytics and charts.

## Features

- Upload workout data
- Track personal records
- Monitor pace progression
- Evaluate efficiency
- Review a training calendar
- Ask AI coach questions
- Evaluate workout history

## How to use

1. Click the **Sync Workouts** button on the top toolbar.
2. Log into the Concept2 logbook with your Concept2 username and password.
3. Review your dashboard.

## Try Chase The Split

Try it live at [Chase The Split](https://chasethesplit.com/).
